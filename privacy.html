Privacy Policy

Last updated: 2026

Overview

This application (“SE Time”) is a work management and time tracking system used by organizations. To use the app, users must log in with an authorized account.

What data we collect

We collect the following types of information:

  • Name and email address
  • Login credentials (securely managed via authentication system)
  • Organization-related data (projects, time registrations, and user assignments)
  • Device and usage information for security and functionality

How we use data

  • To authenticate users and provide secure access
  • To assign users to organizations and projects
  • To store and display time registrations and work data
  • To improve app functionality

Data storage

All data is stored securely in a cloud database hosted on Microsoft Azure. Access is restricted to authenticated users within their organization.

Data sharing

We do not sell or share personal data with third parties. Data is only accessible to users within the same organization where applicable.

Authentication

Login is required to use this application. Authentication is handled securely via a backend API.

Your rights

Users may request deletion or correction of their data by contacting the administrator of their organization.
